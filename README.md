# My Goal Tracker App

🌸 **Welcome to My Goal Tracker App!**

This is a simple mobile application built with **React Native** for tracking and managing goals. You can add, delete, and view your goals in a beautiful, user-friendly interface.

## Features
- Add new goals
- Delete completed or unnecessary goals
- Aesthetic, minimal design with gradient background
- Local data storage (temporarily using React `useState`)

## Technologies Used
- **React Native** for building the app
- **Expo** for easy development and deployment
- **React** for state management

## Getting Started

### Prerequisites

- **Node.js** (Recommended: LTS version)
- **Expo CLI** (to run the app on your phone or simulator)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/your-repository-name.git
````

2. **Navigate to the project directory**

   ```bash
   cd your-repository-name
   ```

3. **Install dependencies**

   ```bash
   npm install
   ```

4. **Start the project**

   ```bash
   expo start
   ```

### Running on your phone

* You can scan the QR code that appears in the terminal after running `expo start` using the **Expo Go** app, available on **iOS** and **Android**.
